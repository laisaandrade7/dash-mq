# 🧠 Decisões do Projeto

## Arquitetura
- Front-end separado do backend
- Backend em PHP
- Dados via JSON/CSV
- Sem API direta na V1

---

## UX
- Desktop-first
- Dashboard operacional (não só visual)
- Foco em leitura rápida e ação

---

## Design
- Dark theme obrigatório
- Estética BI moderno premium
- Evitar qualquer aparência de template comum

---

## Autenticação
- V1: login simples (email/senha)
- Futuro: Google Login

---

## Dados
- Script externo gera JSON
- Dashboard apenas consome

---

## Escopo V1
- Apenas visualização
- Sem automações reais ainda

---

## Ferramentas
- UI/UX Pro Max para front
- Claude Code para construção
- VS Code como ambiente principal
